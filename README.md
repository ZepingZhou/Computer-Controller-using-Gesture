# Computer-Controller-using-Gesture
This repository provides a simple application that controls the computer through gesture recognition.

## Requirements
> + Python >= 3.6
> + PyTorch
> + OpenCV
> + mediapipe

## Installation
> pip install -r requirements.txt

## Demo
> python demo.py

## Features
> + gesture recognition
> + Volume Control
> + Screen Shot
> + QR Code
